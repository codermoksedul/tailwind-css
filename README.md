# tailwind-css

## Taiwlind CSS Install and Projects Building

```
npm init -y
```
```
npm install -D tailwindcss postcss autoprefixer vite
```
```
npx tailwindcss init -p
```

## Change tailwind.config.js
```
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ['*'],
  darkMode: 'media',
  darkMode: 'class',
  theme: {
    extend: {},
  },
  plugins: [],
}
```
```
npm init -y
```
